# personal-bookmark-manager
A sleek, functional web application that allows users to save, categorize, and manage their favorite web links using browser storage.

## 🚀 Features
* **Persistent Storage:** Uses `localStorage` to ensure your bookmarks remain saved even after closing the browser.
* **Category Filtering:** Organizes bookmarks into specific categories (News, Social, etc.) for easy access.
* **Dynamic UI:** Built with a "Single Page App" feel, toggling between the main menu and entry forms.
* **Fully Responsive:** Designed with CSS Flexbox for a clean, centered layout.
* **Validation:** Includes checks for valid name and URL inputs before saving.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the interface.
* **CSS3:** Custom properties (variables), Flexbox layout, and `text-transform` for clean typography.
* **Vanilla JavaScript:** Logic for data persistence, DOM manipulation, and event handling.

## 📂 Project Structure
* `index.html` - The core structure of the application.
* `styles.css` - Custom styling including a dark-themed UI.
* `script.js` - All logic for saving, deleting, and rendering bookmarks.

## 📝 How to Use
1.  Click **"Add Bookmark"** to open the form.
2.  Enter a name, URL, and select a category.
3.  View your bookmarks by selecting a category from the dropdown and clicking **"View Category"**.
4.  To remove a link, select the radio button next to the bookmark and click **"Delete Bookmark"**.
